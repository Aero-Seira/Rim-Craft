# Rim-Craft
> **v0.0.1 WIP**

## 📖 简介

Inspired by **Rimworld** ，将生存管理、科技发展等要素融入 Minecraft 世界。

## 🎮 核心玩法预期

### 🌍 **温饱系统**
- 引入类似群峦的作物与动物机制来实现 Rimworld 的效果
- 食物将会腐烂，食材将变得没那么容易获得
- 温饱问题将上位为生存核心，死亡也不能避免

### 🔬 **非工厂向科技树**
- 基于资源稀有度与物品功能强度的科技树
- 非阶梯式生产，高级材料不会在某一个阶段后变成白菜价
- 弱化自动化玩法，仅在建筑材料等允许大规模生产

### 🌾 **沉浸化QoL**
- 高度统一食物配方与风格
- 复杂化的食物配方
- 多样的装饰模组

### 🛡️ **防御与威胁**
- 类似 Rimworld 的随机性袭击
- 可能偶发的自然灾害
- 更聪明的敌人，战斗机制将强调质量而非数量，同时避免过大规模的破坏
- 你的家园将需要结构规划和防御设施，摊大饼基地再也行不通了



## ⚠️ 开发状态

本项目目前处于 **WIP（开发中）** 阶段，暂无可玩内容。
