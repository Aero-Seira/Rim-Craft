# Rim-Craft
> **v0.0.1 WIP**

## 📖 简介

Inspired by **Rimworld** ，将生存管理、科技发展等要素融入 Minecraft 世界。

## 🎮 核心玩法预期

### 🌍 **环境系统**
- 四季变化与温度系统，影响作物生长和食物存储
- 大型动态天气与自然灾害

### 🔬 **科技进度**
- 基于资源稀有度与物品功能强度的科技树
- 弱化自动化玩法，仅在建筑材料等允许大规模生产

### 🌾 **沉浸化QoL**
- 高度统一食物配方与风格
- 复杂化的食物配方
- 将温饱机制作为生存的核心问题

### 🛡️ **防御与威胁**
- 类似 Rimworld 的随机性袭击
- 更聪明的敌人，战斗机制将强调质量而非数量
- 你的家园将需要结构规划和防御设施，摊大饼基地再也行不通了



## ⚠️ 开发状态

本项目目前处于 **WIP（开发中）** 阶段，暂无可玩内容。
